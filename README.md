# GITIGNORE SCRIPT

To install it:

```bash
    git clone https://github.com/FinaritrAndrianiaina/gitignore-script/
    cd gitignore-script 
    chmod +x ./install.sh
    ./install.sh
    # Verify
    which ignore
```

To use it:

```bash
    ignore macos node rust
```
